# Trepenta Documentation Site

This repository contains the source code for the Trepenta card game documentation website, hosted at [trepenta.ignyos.com](https://trepenta.ignyos.com).

## Purpose

This site provides comprehensive documentation for learning and playing Trepenta, including:
- Core game rules
- Optional house rules (all compatible with each other)
- How to get started

## Technology

Built with pure HTML, CSS, and vanilla JavaScript - no frameworks or libraries required.

## Deployment

The site is automatically deployed via GitHub Pages when changes are pushed to the main branch.
